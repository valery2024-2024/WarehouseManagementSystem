# Звіт по Ітерації 1

## Реалізований функціонал

- створена багатошарова архітектура;
- реалізовані доменні сутності;
- реалізовані репозиторії;
- реалізовані сервіси;
- створений консольний інтерфейс;
- реалізоване управління товарами;
- додані unit-тести.

## Поточна архітектура

Проєкт містить:

- Domain layer;
- Application layer;
- Infrastructure layer;
- Console layer;
- Tests layer.

## Робочі сценарії

1. Додавання товару.
2. Перегляд товарів.

## Ризики

- поки немає підтримки бази даних;
- відсутня авторизація;
- обмежений функціонал замовлень.

## Заплановані покращення

У наступних ітераціях планується:

- додати persistence;
- розширити логіку замовлень;
- додати інтеграційні тести;
- покращити звітність.
